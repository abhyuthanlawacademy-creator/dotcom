# Abhyuthan Law Academy — Website Redesign

A single-page redesign for Abhyuthan Law Academy (judicial-services coaching, Jaipur).

## Files
- `index.html` — the complete, self-contained page (HTML + CSS + JS inline).

## Run
Open `index.html` in any browser, or use a live server:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

Typography uses Google Fonts (Fraunces, Spectral, JetBrains Mono) loaded via `<link>` tags; they render automatically when the page is opened with an internet connection.
